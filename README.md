# 🅾 INSTAGRAM Accounts Video Report — Sentiment Analysis

> Uncovering the emotional pulse of viral Instagram content through NLP-powered comment analysis.

---

## 🌟 Project Overview

This project performs **sentiment analysis on Instagram video comments** across multiple reels, covering themes like mythology, business insights, and wildlife conservation. Using **TextBlob NLP**, it classifies audience reactions as Positive, Negative, or Neutral — giving content creators and brands actionable insights into how their audience truly feels.

---

## 🎯 Objectives

- 🔍 Analyze real-world multilingual Instagram comments (English + Hindi)
- 💡 Classify sentiment per reel: **Positive | Neutral | Negative**
- 📈 Compare audience engagement across different content themes
- 🧠 Demonstrate applied NLP on social media data

## 🧪 Methodology

```
Raw Comments (Multilingual)
        ↓
  TextBlob Polarity Scoring
  (score: -1.0 to +1.0)
        ↓
  Classification:
  Positive  → score > 0
  Neutral   → score = 0
  Negative  → score < 0
        ↓
  Per-Reel Summary Table
```
